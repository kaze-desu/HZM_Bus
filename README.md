## HZM Bus Mordern Project
### Introduction
This project is a modern project for HZM Bus. It is a web application that provides the following features:
- ticket booking
- bus schedule
- profile management
### Technologies
- Frontend: Vue.js
- Backend: Nest.js
- Database: MySQL
